
# Mashroom Portal Demo React Bootstrap App

Plugin for [Mashroom Server](https://www.mashroom-server.com), a **Microfrontend Integration Platform**.

A Demo Microfrontend that uses [React Bootstrap](https://react-bootstrap.netlify.app) components.

It does not deliver any styles itself, but requires a Bootstrap based Portal Theme (like [mashroom-portal-bootstrap-theme](../mashroom-portal-bootstrap-theme)
or the current *Mashroom Portal* default theme) to provide the actual style.

## Usage

If *node_modules/@mashroom* is configured as a plugin path, add **@mashroom/mashroom-portal-demo-react-bootstrap-app** as *dependency*.

Then you can place *Mashroom Portal Demo React Bootstrap App* on any page via *Portal Admin Toolbar*.
