
# Mashroom Portal Demo Shadcn App

Plugin for [Mashroom Server](https://www.mashroom-server.com), a **Microfrontend Integration Platform**.

A Demo Microfrontend that uses [Shadcdn](https://www.shadcn.io) components.

It works best with a Tailwind 4 based theme like the [Mashroom Portal Tailwind Theme](../mashroom-portal-tailwind-theme),
without the Tailwind CSS variables the dark mode might not look correctly.

## Usage

If *node_modules/@mashroom* is configured as a plugin path, add **@mashroom/mashroom-portal-demo-react-shadcn-app** as *dependency*.

Then you can place *Mashroom Portal Demo React Shadcn App* on any page via *Portal Admin Toolbar*.
