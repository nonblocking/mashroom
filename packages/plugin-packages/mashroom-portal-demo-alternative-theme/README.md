
# Mashroom Portal Demo Alternative Theme

Plugin for [Mashroom Server](https://www.mashroom-server.com), a **Microfrontend Integration Platform**.

This is an alternative demo theme for the [Mashroom Portal](../mashroom-portal).

This theme demonstrates how to create a **type safe** theme with *express-react-views* and *TypeScript*.

## Usage

If *node_modules/@mashroom* is configured as a plugin path, add **@mashroom/mashroom-portal-demo-alternative-theme** as *dependency*.
