
# Mashroom Portal Default Layouts

Plugin for [Mashroom Server](https://www.mashroom-server.com), a **Microfrontend Integration Platform**.

This plugin adds some default layouts to the [Mashroom Portal](../mashroom-portal).

## Usage

If *node_modules/@mashroom* is configured as a plugin path, add **@mashroom/mashroom-portal-default-layouts** as *dependency*.
